Run Your First Job

note

This topic is intended to provide an overview of running jobs in the Integration Manager. If you haven't yet run a job, some of the steps in this process might require further explanation. Refer to the links provided.

Step 1: Create a Configuration​

Refer to Creating Configurations.

  1. Select the Configurations tab:

  2. Click .

  3. The Import Configuration page is displayed:

  4. Enter a unique Name, then do one of the following:

    • Click Local System, select a configuration file (.djar,.rtc,.process,.ip.xml,.tf.xml,.sqlite), then click Import.
    • Click Integration Files. In the File Manager, select a package, then click Select Package. Should we describe the File Manager more fully?

  5. The configuration file is listed. Click Import.

  6. If successful, the Configuration Details page opens with the message "Integration Succcessfully Imported!". What to do if not? What reasons for failure?

Step 4: Run the Job​

  1. Click Run Configuration.
  2. Navigate to the Jobs tab to view job progress and log file.
