1. Click &nbsp;![the Add Macro button](/img/icons/Add-Macro-Button.png).
2. A row is added to the Macros list with the default name “Macro #”:

    ![Add Another Macro](/img/Add-Another-Macro.png)

3. Edit the name as desired and press Enter.
4. Click the empty space in the **Value** column, enter the macro value, and press Enter.

    ![Edit Value](/img/Add-Macro-Value.png)

5. Click the empty space in the **Description** column, enter a description, and press Enter.

    ![Edit Description](/img/Add-Macro-Description.png)
